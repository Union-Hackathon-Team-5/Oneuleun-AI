# 오늘은 (Today) - 프로젝트 컨텍스트

## 서비스 개요
독거노인이 영상으로 일상을 기록하면, AI가 감정과 상태를 분석해 보호자에게 전달하는 돌봄 서비스

## 핵심 사용자
1. **독거노인**: 영상 편지를 촬영하는 사용자
2. **보호자**: 독거노인의 상태를 모니터링하는 사용자 (가족, 요양보호사 등)

## 주요 플로우

### 독거노인 - 영상 편지 촬영
1. 촬영 시작
2. TTS가 질문 ("오늘은 뭐 하셨어요?", "오늘 기분은 어떠세요?" 등)
3. 사용자가 자유롭게 답변하며 대화
4. 촬영 중 실시간으로 기록:
   - 표정 (감정 상태)
   - 음성 (톤, 속도)
   - 대화 내용
5. 촬영 종료

### AI 분석
촬영 완료 후 자동으로:
- 표정 데이터로 감정 상태 분석 (긍정/부정/불안/우울 등)
- 음성 데이터로 심리 상태 파악
- 대화 내용 텍스트화 및 키워드 추출
- 건강/안전 관련 이슈 감지 (예: "어지러워요", "넘어졌어요", "밥을 안 먹었어요")
- 종합 분석 리포트 생성

### 보호자 - 대시보드
보호자가 확인하는 정보:

#### 1. 영상 편지
- 독거노인이 촬영한 영상
- AI가 요약한 대화 내용
- 주요 언급 사항

#### 2. 감정 상태 트렌드
- 기간별 감정 변화 (일별, 주별, 월별)
- 감정 그래프 시각화
- 이전 대비 변화율

#### 3. 상담 내용 분석
- 자주 언급하는 주제
- 건강 관련 언급 사항
- 사회 활동 여부 (외출, 대인관계 등)
- 특이사항 및 주의 필요 내용

#### 4. 추천 조치
AI가 현재 상태를 분석해 제안:
- **긴급**: 즉시 연락/방문 필요
- **주의**: 며칠 내 확인 필요
- **권장**: 이번 주 내 케어 필요
- **정상**: 현재 상태 양호

구체적 조치 예시:
- "감정 상태가 3일 연속 저하되었습니다. 전화 또는 방문을 권장합니다"
- "식사를 거르는 언급이 증가했습니다. 식사 여부를 확인해주세요"
- "외출 빈도가 감소했습니다. 야외 활동을 제안해보세요"

## 데이터 구조 개념

### 영상 편지 (Video Letter)
- 촬영 일시
- 영상 파일
- 촬영 길이
- 상태 (처리 중, 분석 완료)

### 분석 데이터 (Analysis Data)
- 감정 점수 (긍정, 부정, 불안, 우울 등의 수치)
- 주요 키워드 목록
- 건강/안전 이슈 목록
- 전체 요약문
- 위험도 레벨

### 사용자 관계 (User Relationship)
- 독거노인 1명 : 보호자 N명 (1:N)
- 보호자는 여러 독거노인 담당 가능 (N:M)

## 용어 정의
- **영상 편지**: 독거노인이 촬영하는 일상 기록 영상
- **상담**: 영상 편지 촬영 행위 자체 (실제 상담사가 있는 것이 아님)
- **감정 상태**: AI가 분석한 현재 심리/감정 상태
- **조치**: 보호자가 취해야 할 행동 제안
- **트렌드**: 시간에 따른 감정/상태 변화 추이

## 주요 기능 요구사항

### 독거노인 앱
- 매우 간단한 인터페이스 (버튼 크게, 글씨 크게)
- 음성 안내 강화 (TTS)
- 한 번의 터치로 촬영 시작
- 자동 업로드

### 보호자 대시보드
- 여러 독거노인의 상태를 한눈에 파악
- 알림 기능 (새 영상, 위험 감지 시)
- 데이터 시각화 (그래프, 차트)
- 기간별 필터링 및 비교

### AI 분석
- 표정 인식
- 음성 감정 분석
- 음성을 텍스트로 변환 (STT)
- 텍스트 감정 분석 및 키워드 추출
- 이상 패턴 감지 (평소 대비 변화)

## 설계 고려사항

### 사용자 경험
- 고령자 친화적: 복잡한 조작 없이 직관적으로
- 보호자 효율성: 여러 사람을 빠르게 모니터링
- 신뢰성: 정확한 분석으로 불필요한 불안 최소화

### 데이터 처리
- 영상은 촬영 후 자동 업로드
- 분석은 백그라운드에서 자동 진행
- 분석 완료 시 보호자에게 알림

### 알림 우선순위
1. 긴급 (위험 감지) - 즉시 알림
2. 주의 (상태 변화) - 중요 알림
3. 정보 (새 영상) - 일반 알림

## 주요 시나리오

### 시나리오 1: 일상적인 사용
1. 독거노인이 저녁 식사 후 영상 편지 촬영
2. "오늘은 뭐 하셨어요?" 질문에 "손주가 다녀갔어요, 기분이 좋아요" 답변
3. AI가 긍정 감정(85%) 분석, "손주", "기분 좋음" 키워드 추출
4. 보호자에게 "정상" 상태로 리포트 전달

### 시나리오 2: 위험 감지
1. 독거노인이 영상 촬영 중 "어제 넘어졌어요, 허리가 아파요" 언급
2. AI가 "낙상", "통증" 키워드 감지, 위험도 "긴급" 판정
3. 보호자에게 즉시 푸시 알림 + "병원 방문 확인 필요" 조치 제안

### 시나리오 3: 감정 변화 추적
1. 1주일간 독거노인의 감정 점수가 지속 하락 (80→70→60→50)
2. "혼자 있으니 외로워요", "요즘 잠을 못 자요" 등 부정적 언급 증가
3. 보호자 대시보드에 감정 하락 트렌드 표시
4. AI가 "주의" 단계로 분류, "주 2회 이상 연락 권장" 조치 제안

## 개발 시 참고사항
- 이 서비스는 실시간 상담이 아닌 **비동기 모니터링** 서비스
- AI는 **보조 도구**이며, 최종 판단은 보호자가 수행
- 독거노인의 **일상 루틴** 형성이 중요 (매일 같은 시간에 촬영 유도)
- 보호자가 **부담 없이** 지속적으로 확인할 수 있어야 함
